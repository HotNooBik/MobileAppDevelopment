Было реализовано следующее:

1. Добавлена карта (OpenStreetMap) с расположением Столба на экране его описания;
2. Подключен API (api.open-meteo.com) для получения данных о погоде за день (за следующие 14 дней, начиная от текущего) и для вычисления степени благоприятности дня для посещения Столбов с выводом этих всех данных в календарь.

Логика работы с API реализована через сервис.
